Comparison Functions

clamp()·min()·max()

clamp() — Fluid typografie — nadpis se plynule mění s šířkou okna

clamp(minimum, preferred, maximum)
nejmenší povolená hodnota ideální hodnota (fluid) největší povolená hodnota
font-size: clamp(1.25rem, 4vw, 3rem) plynule roste s oknem

Fluid nadpis

font-size: var(--font-size-xl) fixní velikost

Fixní nadpis

Zmenšuj nebo zvětšuj okno prohlížeče — fluid nadpis se mění plynule, fixní zůstává stejný.

min() — Šířka kontejneru — vrátí nejmenší z hodnot

width: min(100%, 640px) 100 % nebo maximálně 640 px — vždy menší hodnota
min(100%, 640px)
max-width: 640px ekvivalentní zápis — ale min() přijme i více argumentů
max-width: 640px
Proč min()?

Na rozdíl od max-width přijme min() libovolný počet argumentů a může kombinovat různé jednotky — například min(50vw, 400px, 100%).

max() — Garantované minimum — vrátí největší z hodnot

padding: max(var(--base-2), 3vw) padding roste s viewportem, ale nikdy neklesne pod minimum
Obsah s fluid paddingem
font-size: max(14px, 1.5vw) nikdy nečitelně malé — minimum 14 px zaručeno

Tento text je nikdy menší než 14 px, ale roste s šířkou okna díky 1.5vw.

border-radius: max(4px, 1vw) zaoblení roste s viewportem, nikdy neklesne pod 4 px
Proč max()?

Používá se jako "spodní zarážka" — garantuje minimální hodnotu, zatímco umožňuje fluid chování nahoru. Ideální pro dostupnost (čitelnost textu) a dostatečné klepací plochy.

clamp() — Responsivní karta — gap, padding, radius i font plynule reagují na viewport

gap clamp(var(--base-2), 3vw, var(--base-6))
padding clamp(var(--base-2), 2vw, var(--base-5))
border-radius clamp(4px, 1vw, 16px)
font-size clamp(0.875rem, 0.7rem + 0.6vw, 1.125rem)
A
Karta první
Fluid gap zaručuje čitelné mezery na každé obrazovce.
B
Karta druhá
Na mobilu minimum, na desktopu maximum, mezi tím plynulý přechod.
C
Karta třetí
Žádné breakpointy — clamp() zvládne vše sám.
Zmenšuj nebo zvětšuj okno — sleduj jak se plynule mění mezery, zaoblení, padding i velikost textu karet.

clamp() — Fluid type scale — proporcionální typografická škála bez breakpointů

--step-3: clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem) každá úroveň škály je fluid — h1, h2 i body text mění velikost společně

Moderní typografie bez breakpointů

Fluid škála zachovává proporce na každé šířce okna

Každá úroveň je definovaná samostatnou clamp() hodnotou — minimum, preferovaná fluid hodnota a maximum. Nadpisy a tělo textu tak rostou společně a jejich vzájemný poměr zůstává konzistentní od mobilního rozlišení až po wide screen.

Zmenšuj nebo zvětšuj okno — všechny tři úrovně se mění plynule a zachovávají vzájemné proporce.
Fluid type scale — čtyři kroky

--step-0 (body): clamp(0.875rem, 0.8rem + 0.4vw, 1rem)
--step-1 (h3): clamp(1.125rem, 1rem + 0.6vw, 1.375rem)
--step-2 (h2): clamp(1.375rem, 1.2rem + 0.9vw, 1.875rem)
--step-3 (h1): clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem) 

/* ── CSS Comparison Functions ────────────────────────────────────────────── */

/* clamp(minimum, preferred, maximum)
   Vrátí preferred, ale nikdy méně než minimum a nikdy více než maximum.
   Ideální pro fluid typografii a spacing. */

.fluid-heading {
  font-size: clamp(1.25rem, 4vw, 3rem);
  /* Na úzkém okně: 1.25rem | Fluid střed: 4vw | Na širokém: 3rem */
}

.fluid-gap-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: clamp(var(--base-2), 3vw, var(--base-6));
}

/* Responsivní karta — gap, padding, border-radius i font v jednom */
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: clamp(var(--base-2), 3vw, var(--base-6));
}

.card {
  padding:       clamp(var(--base-2), 2vw,  var(--base-5));
  border-radius: clamp(4px,           1vw,  16px);
  font-size:     clamp(0.875rem, 0.7rem + 0.6vw, 1.125rem);
}


/* ── min() ───────────────────────────────────────────────────────────────── */

/* min(a, b, …) — vrátí nejmenší hodnotu ze seznamu.
   Výborné pro omezení šířky: buď 100 % nebo max. 640 px — co je menší. */

.container {
  width: min(100%, 640px);
  /* Ekvivalentní: max-width: 640px — ale min() přijme více argumentů */
}

/* Kombinace více jednotek — není možné s max-width */
.sidebar {
  width: min(50vw, 320px, 100%);
}


/* ── max() ───────────────────────────────────────────────────────────────── */

/* max(a, b, …) — vrátí největší hodnotu ze seznamu.
   Výborné jako "spodní zarážka": garantuje minimum, fluid nahoru. */

.card {
  padding: max(var(--base-2), 3vw);
  /* Na úzkém okně: var(--base-2) | Na širokém: 3vw (větší) */
}

.body-text {
  font-size: max(14px, 1.5vw);
  /* Nikdy pod 14 px — zaručuje čitelnost na malých obrazovkách */
}

/* border-radius — fluid zaoblení s garantovaným minimem */
.btn {
  border-radius: max(4px, 1vw);
  /* Na mobilu: 4 px | Na desktopu: 1vw (větší) — vždy alespoň trochu zaoblené */
}

/* clamp() pro radius: ohraničí i maximum — hodí se např. na karty */
.card {
  border-radius: clamp(4px, 1vw, 16px);
  /* Pod 4 px nikdy, nad 16 px nikdy, mezi tím fluid */
}


/* ── Kombinace ────────────────────────────────────────────────────────────── */

/* clamp() je zkratka pro max(min, min(preferred, max)) */
font-size: clamp(1rem, 2.5vw, 2rem);
/* === */
font-size: max(1rem, min(2.5vw, 2rem));


/* ── Fluid type scale ────────────────────────────────────────────────────── */

/* Ucelená škála — proporce zůstávají konzistentní na každé šířce okna */
:root {
  --step-0: clamp(0.875rem, 0.8rem + 0.4vw, 1rem);    /* body */
  --step-1: clamp(1.125rem, 1rem + 0.6vw, 1.375rem);  /* h3   */
  --step-2: clamp(1.375rem, 1.2rem + 0.9vw, 1.875rem);/* h2   */
  --step-3: clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem);  /* h1   */
}

h1   { font-size: var(--step-3); }
h2   { font-size: var(--step-2); }
h3   { font-size: var(--step-1); }
body { font-size: var(--step-0); }
