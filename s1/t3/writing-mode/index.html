Logické vlastnosti a writing-mode

writing-mode

Fyzické vlastnosti jako padding-left nebo padding-right jsou vázané na viewport, nikoli na směr textu. U vertikálního textu (writing-mode: vertical-rl) se osy otočí — to, co bylo "vlevo", je teď nahoře. Logické vlastnosti se otočí automaticky.

Fyzické vlastnosti
Štítek A
Delší popis
/* Fyzické — otočení osy je matoucí */
.tag {
  writing-mode: vertical-rl;
  padding-left: 1rem;   /* → teď nahoře */
  padding-right: 1rem;  /* → teď dole */
  border-left: 4px solid; /* → teď nahoře */
}
Logické vlastnosti
Štítek A
Delší popis
/* Logické — otočí se automaticky ✓ */
.tag {
  writing-mode: vertical-rl;
  padding-block: 1rem;    /* → podél textu */
  padding-inline: 0.5rem; /* → napříč textu */
  border-inline-start: 4px solid; /* → začátek textu */
}
U fyzické karty je border-left nahoře — protože po otočení writing-mode je "vlevo" fyzicky nahoře. Logická border-inline-start vždy označuje začátek řádku textu, ať už je směr čtení jakýkoli.

writing-mode a logické vlastnosti

/* Po otočení writing-mode se fyzické osy přeuspořádají:
   vertical-rl → "vlevo" = nahoře, "vpravo" = dole                    */

/* Fyzicky — špatně, padding-left jde nahoru místo doleva */
.tag-bad {
  writing-mode: vertical-rl;
  padding-left: 1rem;    /* teď je to nad textem */
  border-left: 4px solid;/* teď je to nad textem */
}

/* Logicky — správně, padding-block jde podél textu */
.tag-good {
  writing-mode: vertical-rl;
  padding-block: 1rem;          /* = podél osy textu */
  padding-inline: 0.5rem;       /* = napříč osy textu */
  border-inline-start: 4px solid; /* = začátek řádku textu */
}
