S1 – Foundations Refresh

Physical → Logical mapping

margin-inline·padding-block

Přehled nejpoužívanějších fyzických vlastností a jejich logických protějšků.

Fyzická vlastnost Logická vlastnost Popis
width inline-size Rozměr podél osy inline (→)
height block-size Rozměr podél osy blokové (↓)
margin-top / margin-bottom margin-block Odsazení v blokové ose (zkratka pro obě)
margin-left / margin-right margin-inline Odsazení v inline ose (zkratka pro obě)
padding-top padding-block-start Vnitřní odsazení začátku blokové osy
padding-bottom padding-block-end Vnitřní odsazení konce blokové osy
top / right / bottom / left inset Zkratka pro všechny čtyři strany
left inset-inline-start Začátek inline osy (v LTR = vlevo)
border-left border-inline-start Rámeček začátku inline osy
text-align: left text-align: start Zarovnání k začátku inline osy

Velikosti

.box {
  width: 200px;          /* → */  inline-size: 200px;
  height: 100px;         /* → */  block-size: 100px;
}

Odsazení

.card {
  margin-top: 1rem;      /* → */  margin-block-start: 1rem;
  margin-bottom: 1rem;   /* → */  margin-block-end: 1rem;
  margin-left: 2rem;     /* → */  margin-inline-start: 2rem;

  /* Zkratky (obě strany najednou) */
  margin-block: 1rem;             /* = margin-top + margin-bottom */
  margin-inline: 2rem;            /* = margin-left + margin-right */

  padding-block-start: 1.5rem;    /* = padding-top */
  border-inline-start: 3px solid; /* = border-left */
  text-align: start;              /* = text-align: left v LTR */
}
