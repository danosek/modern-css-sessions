Logical Properties

margin-inline·padding-block·inset
Živá ukázka — fyzické vs. logické vlastnosti

Tlačítko přepíná směr čtení (dir) na celém demo kontejneru. Logická verze se automaticky překlopí do RTL, fyzická ne.

Aktuální směr: LTR
Fyzické vlastnosti
physical

Název položky

Tato karta používá margin-left, padding-top a border-left. Při přepnutí do RTL se tyto vlastnosti nepřeklopí — levý okraj zůstane vlevo.

fyzická nepřeklápí se 
/* Fyzické vlastnosti */
.card {
  margin-left: 1rem;
  padding-top: 1.5rem;
  border-left: 3px solid;
  text-align: left;
}
Logické vlastnosti
logical

Název položky

Tato karta používá margin-inline-start, padding-block-start a border-inline-start. Při přepnutí do RTL se automaticky překlopí na správnou stranu.

logická překlopí se ✓ 
/* Logické vlastnosti */
.card {
  margin-inline-start: 1rem;
  padding-block-start: 1.5rem;
  border-inline-start: 3px solid;
  text-align: start;
}
Physical → Logical mapping

Přehled nejpoužívanějších fyzických vlastností a jejich logických protějšků.

Fyzická vlastnost Logická vlastnost Popis
width inline-size Rozměr podél osy inline (→)
height block-size Rozměr podél osy blokové (↓)
margin-top / margin-bottom margin-block Odsazení v blokové ose (zkratka pro obě)
margin-left / margin-right margin-inline Odsazení v inline ose (zkratka pro obě)
padding-top padding-block-start Vnitřní odsazení začátku blokové osy
padding-bottom padding-block-end Vnitřní odsazení konce blokové osy
top / right / bottom / left inset Zkratka pro všechny čtyři strany
left inset-inline-start Začátek inline osy (v LTR = vlevo)
border-left border-inline-start Rámeček začátku inline osy
text-align: left text-align: start Zarovnání k začátku inline osy
Zkratka inset

Vlastnost inset nahrazuje všechny čtyři poziční vlastnosti najednou a funguje stejně jako margin či padding co do pořadí hodnot.

Fyzicky (4 řádky)
/* Fyzicky — 4 vlastnosti */
.overlay {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}
fyzické
  • top: 0
  • right: 0
  • bottom: 0
  • left: 0
→
Logicky (1 řádek)
/* Logicky — zkratka inset */
.overlay {
  position: absolute;
  inset: 0;
}

/* nebo s různými hodnotami */
.panel {
  inset: 1rem 0 0 1rem;
}
inset: 0
  • inset: 0 ← vše najednou
/* ── Logical Properties ──────────────────────────────────────────────────── */

/* Fyzické vlastnosti jsou vázané na viewport (top/right/bottom/left) */
/* Logické vlastnosti jsou vázané na směr psaní (block / inline osy) */

/* Inline osa → v LTR = vodorovná (←→), v RTL se otočí */
/* Block osa  ↓ v LTR = svislá (↑↓), nemění se */


/* ── Velikosti ────────────────────────────────────────────────────────────── */

.box {
  width: 200px;          /* → */  inline-size: 200px;
  height: 100px;         /* → */  block-size: 100px;
}


/* ── Odsazení ─────────────────────────────────────────────────────────────── */

.card {
  margin-top: 1rem;      /* → */  margin-block-start: 1rem;
  margin-bottom: 1rem;   /* → */  margin-block-end: 1rem;
  margin-left: 2rem;     /* → */  margin-inline-start: 2rem;

  /* Zkratky (obě strany najednou) */
  margin-block: 1rem;             /* = margin-top + margin-bottom */
  margin-inline: 2rem;            /* = margin-left + margin-right */

  padding-block-start: 1.5rem;    /* = padding-top */
  border-inline-start: 3px solid; /* = border-left */
  text-align: start;              /* = text-align: left v LTR */
}


/* ── Pozicování — inset ──────────────────────────────────────────────────── */

/* Fyzicky: */
.overlay {
  position: absolute;
  top: 0; right: 0; bottom: 0; left: 0;
}

/* Logicky — zkratka inset: */
.overlay {
  position: absolute;
  inset: 0;           /* = top right bottom left = 0 */
}

/* inset funguje jako margin/padding shorthand: */
.panel {
  inset: 1rem 2rem;         /* block: 1rem, inline: 2rem */
  inset-inline-start: 0;    /* = left v LTR, right v RTL */
}


/* ── Proč záleží? ─────────────────────────────────────────────────────────── */

/* RTL jazyky (arabština, hebrejština) nebo vertical writing (japonština) —
   logické vlastnosti se automaticky přizpůsobí bez změny CSS.
   Fyzické vlastnosti zůstanou vždy na stejné straně viewportu.        */
