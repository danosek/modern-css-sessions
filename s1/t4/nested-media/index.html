@media uvnitř pravidla

Zmenši okno prohlížeče a sleduj změnu:

@media

Responzivní nadpis

Tento blok mění layout, velikost písma i padding — vše definováno uvnitř jednoho CSS pravidla.

Desktop Mobil 
.fluid-hero {
  display: flex;
  flex-direction: row;       /* desktop: řádek */
  gap: var(--base-4);
  padding: var(--base-5);

  /* @media přímo uvnitř pravidla! */
  @media (max-width: 600px) {
    flex-direction: column;  /* mobil: sloupec */
    padding: var(--base-3);
  }

  & .fluid-hero__title {
    font-size: var(--font-size-xxl);

    @media (max-width: 600px) {
      font-size: var(--font-size-xl);  /* menší na mobilu */
    }
  }
}
