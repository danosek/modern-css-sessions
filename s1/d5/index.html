S1 – Foundations Refresh

@layer — správa kaskády

@layer·layer()·revert-layer

Živá ukázka — pořadí vrstev bije specificitu selektorů

@layer reset Normalize, základní reset
@layer base Typografie, barvy, tokeny
@layer components .btn, .card, .input…
@layer utilities .mt-4, .text-center, .hidden…
bez vrstvy (unlayered) Inline styly, přepsání ad hoc

↑ Vyšší vrstva = vyšší priorita — bez ohledu na specificitu selektoru

Hlavní princip — pořadí vrstvy bije specificitu

@layer utilities vrstva vítězí navzdory nižší specificitě
.btn-primary components (0,1,0) primary
.bg-brand utilities (0,1,0) ← vrstva je výše secondary ✓
bez @layer vyšší specificita vítězí (klasická kaskáda)
.btn-primary-nl specificita (0,1,0) — stejná primary ✓ (pozdější v kódu)
.bg-brand-nl specificita (0,1,0) — dřívější v kódu secondary

Praktické příklady

@import layer() Third-party knihovna v nejnižší vrstvě
Knihovna (reset)

Styly jsou "uzamčeny" do vrstvy reset — nemohou ovlivnit naše komponenty.

Naše komponenta (components)

Přebije knihovnu bez !important — prostě je v nižší vrstvě.

revert-layer Vrácení hodnoty na předchozí vrstvu

background: revert-layer vrátí hodnotu na to, co definovala nižší vrstva — bez znalosti konkrétní hodnoty.

@layer { … } Anonymní vrstva — vždy nižší než pojmenované
bez vrstvy (unlayered) priorita 4 — nejvyšší
@layer theme priorita 3 ✓
@layer base priorita 2
@layer { … } — anonymní priorita 1 — vždy nejnižší
theme (pojmenovaná) base (pojmenovaná) @layer { } anonymní

Deklarace @layer base, theme; fixuje pořadí — theme vždy bije base, bez ohledu na pořadí v kódu. Anonymní vrstva @layer { … } je vždy pod všemi pojmenovanými.

/* Pořadí deklarováno dopředu — poté lze @layer naplnit kamkoliv v kódu */
@layer base, theme;

/* theme > base — přestože @layer base je v kódu zapsán dřív */
@layer theme { .btn { background: var(--surface-brand-primary-strong); } }
@layer base  { .btn { background: var(--surface-main-dim); } }

/* Anonymní vrstva — VŽDY nižší než pojmenované, i když je v kódu nakonec */
@layer { .btn { background: red; } }   /* nikdy nevyhraje */
/* Deklaruj pořadí vrstev na začátku CSS */
@layer reset, base, components, utilities;

/* Naplň vrstvy */
@layer reset {
  * { box-sizing: border-box; margin: 0; }
}

@layer components {
  .btn-primary {
    background: var(--surface-brand-primary-strong);  /* specificita: 0,1,0 */
  }
}

@layer utilities {
  .bg-secondary {
    background: var(--surface-brand-secondary-strong); /* specificita: 0,1,0 */
  }
  /* .bg-secondary VÍTĚZÍ — utilities je výše než components */
}

/* Third-party knihovna do nejnižší vrstvy */
@import url("bootstrap.css") layer(vendor);
/* Naše styly vždy přebijí vendor — bez !important */

/* Vrácení hodnoty na předchozí vrstvu */
@layer components {
  .btn-ghost {
    background: revert-layer;  /* vezme hodnotu z vrstvy 'base', ne components */
  }
}
