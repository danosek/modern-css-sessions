clamp() — fluid type scale

Proporcionální typografická škála bez breakpointů

font-size·type-scale
--step-3: clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem) každá úroveň škály je fluid — h1, h2 i body text mění velikost společně

Moderní typografie bez breakpointů

Fluid škála zachovává proporce na každé šířce okna

Každá úroveň je definovaná samostatnou clamp() hodnotou — minimum, preferovaná fluid hodnota a maximum. Nadpisy a tělo textu tak rostou společně a jejich vzájemný poměr zůstává konzistentní od mobilního rozlišení až po wide screen.

Zmenšuj nebo zvětšuj okno — všechny tři úrovně se mění plynule a zachovávají vzájemné proporce.
Fluid type scale — čtyři kroky

--step-0 (body): clamp(0.875rem, 0.8rem + 0.4vw, 1rem)
--step-1 (h3): clamp(1.125rem, 1rem + 0.6vw, 1.375rem)
--step-2 (h2): clamp(1.375rem, 1.2rem + 0.9vw, 1.875rem)
--step-3 (h1): clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem)

Fluid type scale

/* Ucelená škála — proporce zůstávají konzistentní na každé šířce okna */
:root {
  --step-0: clamp(0.875rem, 0.8rem + 0.4vw, 1rem);    /* body */
  --step-1: clamp(1.125rem, 1rem + 0.6vw, 1.375rem);  /* h3   */
  --step-2: clamp(1.375rem, 1.2rem + 0.9vw, 1.875rem);/* h2   */
  --step-3: clamp(1.75rem, 1.4rem + 1.7vw, 2.75rem);  /* h1   */
}

h1   { font-size: var(--step-3); }
h2   { font-size: var(--step-2); }
h3   { font-size: var(--step-1); }
body { font-size: var(--step-0); }
