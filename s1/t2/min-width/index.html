min() — šířka kontejneru

Vrátí nejmenší z hodnot

width·max-width
width: min(100%, 640px) →
100 % nebo maximálně 640 px — vždy menší hodnota
min(100%, 640px)
max-width: 640px ekvivalentní zápis — ale min() přijme i více argumentů
max-width: 640px
Proč min()?

Na rozdíl od max-width přijme min() libovolný počet argumentů a může kombinovat různé jednotky — například min(50vw, 400px, 100%).

min()

/* min(a, b, …) — vrátí nejmenší hodnotu ze seznamu.
   Výborné pro omezení šířky: buď 100 % nebo max. 640 px — co je menší. */

.container {
  width: min(100%, 640px);
  /* Ekvivalentní: max-width: 640px — ale min() přijme více argumentů */
}

/* Kombinace více jednotek — není možné s max-width */
.sidebar {
  width: min(50vw, 320px, 100%);
}
