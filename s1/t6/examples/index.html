.card:has(.card__img) Layout se automaticky přizpůsobí obsahu
◫
S obrázkem — horizontální grid

Karta obsahuje .card__img, proto :has(.card__img) přepne layout na grid-template-columns: 100px 1fr.

Bez obrázku — textový blok

Karta neobsahuje .card__img. Selektor :has() se neaplikuje, karta zůstane jako jednoduchý blok.

Grid sám zjistí kolik má přímých potomků — žádný JS, žádné třídy přidávané ručně. :has(> .qq-item:nth-child(3)) se aplikuje, jakmile existuje třetí přímý potomek.

/* 1–2 položky: jeden sloupec (výchozí) */
.qq-grid { display: grid; grid-template-columns: 1fr; gap: var(--base-2); }

/* 3+ položky: dva sloupce */
.qq-grid:has(> .qq-item:nth-child(3)) {
  grid-template-columns: 1fr 1fr;
}

/* 5+ položky: tři sloupce */
.qq-grid:has(> .qq-item:nth-child(5)) {
  grid-template-columns: 1fr 1fr 1fr;
}
