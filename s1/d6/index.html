S obrázkem — horizontální grid

Karta obsahuje .card__img, proto :has(.card__img) přepne layout na grid-template-columns: 100px 1fr.

Bez obrázku — textový blok

Karta neobsahuje .card__img. Selektor :has() se neaplikuje, karta zůstane jako jednoduchý blok.

Grid sám zjistí kolik má přímých potomků — žádný JS, žádné třídy přidávané ručně. :has(> .qq-item:nth-child(3)) se aplikuje, jakmile existuje třetí přímý potomek.

/* 1–2 položky: jeden sloupec (výchozí) */
.qq-grid { display: grid; grid-template-columns: 1fr; gap: var(--base-2); }

/* 3+ položky: dva sloupce */
.qq-grid:has(> .qq-item:nth-child(3)) {
  grid-template-columns: 1fr 1fr;
}

/* 5+ položky: tři sloupce */
.qq-grid:has(> .qq-item:nth-child(5)) {
  grid-template-columns: 1fr 1fr 1fr;
}

:is() a :where() — skupinové selektory

:is() — specificita = max(argumentů)
article :is(h1, h2, h3) { color: … } ≡ article h1, article h2, article h3 { … }

Nadpis druhé úrovně

Odstavec — není ve skupině :is()

Nadpis třetí úrovně

Specificita odpovídá nejsilnějšímu argumentu. :is(#id, .class) má specificitu ID selektoru pro všechny pravidla ve skupině.

:where() — specificita vždy = 0
:where(.box) { background: violet; } specificita (0,0,0) → přepsatelné čímkoliv
Výchozí styl přes :where()
Přepsán třídou .where-box--override
funguje, protože :where() má specificitu 0

Ideální pro reset / base styly — jakýkoliv selektor je přepíše bez !important. Používá se v design systémech pro přepisovatelné výchozí hodnoty.

/* :has() — rodičovský selektor */

/* Celá řádka reaguje na stav checkboxu uvnitř — čistý CSS */
.task:has(input:checked) {
  opacity: 0.5;
}
.task:has(input:checked) .task-name {
  text-decoration: line-through;
}

/* Validace formuláře bez JS */
.field:has(input:user-invalid) .field-label { color: var(--status-error); }
.field:has(input:user-invalid) .field-input  { border-color: var(--status-error); }
.field:has(input:user-valid)   .field-label  { color: var(--status-success); }
.field:has(input:user-valid)   .field-input  { border-color: var(--status-success); }

/* Navigace: zvýrazni sekci, když je potomek aktivní */
.nav-section:has(.is-active) {
  background: var(--surface-brand-primary-subtle);
  border-left: 3px solid var(--surface-brand-primary-strong);
}

/* Layout se mění podle přítomnosti potomka */
.card:has(.card__img) {
  display: grid;
  grid-template-columns: 100px 1fr;
  gap: var(--base-3);
  align-items: start;
}

/* :is() — zkratka (specificita = nejvyšší argument) */
article :is(h1, h2, h3, h4) { font-family: var(--font-stack-headings); }

/* :where() — jako :is(), ale specificita = 0 → vždy přepsatelné */
:where(button, input, select) { border-radius: var(--radius); }
