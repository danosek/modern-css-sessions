background: revert-layer vrátí hodnotu na to, co definovala nižší vrstva — bez znalosti konkrétní hodnoty.

@layer { … } Anonymní vrstva — vždy nižší než pojmenované
bez vrstvy (unlayered) priorita 4 — nejvyšší
@layer theme priorita 3 ✓
@layer base priorita 2
@layer { … } — anonymní priorita 1 — vždy nejnižší
theme (pojmenovaná) base (pojmenovaná) @layer { } anonymní

Deklarace @layer base, theme; fixuje pořadí — theme vždy bije base, bez ohledu na pořadí v kódu. Anonymní vrstva @layer { … } je vždy pod všemi pojmenovanými.

/* Pořadí deklarováno dopředu — poté lze @layer naplnit kamkoliv v kódu */
@layer base, theme;

/* theme > base — přestože @layer base je v kódu zapsán dřív */
@layer theme { .btn { background: var(--surface-brand-primary-strong); } }
@layer base  { .btn { background: var(--surface-main-dim); } }

/* Anonymní vrstva — VŽDY nižší než pojmenované, i když je v kódu nakonec */
@layer { .btn { background: red; } }   /* nikdy nevyhraje */
