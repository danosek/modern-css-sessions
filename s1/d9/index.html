reading-flow

reading-flow·reading-order
ⓘ Vlastnost reading-flow vyžaduje Chrome 137+. Demo níže vizualizuje pořadí focusu v obou scénářích.
Živá ukázka — zmáčkni Tab a sleduj pořadí focusu (DOM pořadí = A→B→C→D→E→F):
Klikni na tlačítko nebo zmáčkni Tab…
Problém a řešení:
Bez reading-flow Tab = DOM pořadí
ADOM 1
BDOM 2
CDOM 3
DDOM 4
EDOM 5
FDOM 6

Tab: A→B→C→D→E→F (DOM pořadí)
Vizuálně: D→B→F→A→C→E
→ skáče po obrazovce chaoticky

reading-flow: grid-order Tab = vizuální pořadí
Avis. 4
Bvis. 2
Cvis. 6
Dvis. 1
Evis. 5
Fvis. 3

Tab: D→B→F→A→C→E (vizuální pořadí)
→ plynule zleva doprava, shora dolů
→ screen readery i klávesnice fungují přirozeně

Kanban board — úkoly jsou v DOM seřazeny po sloupcích (TODO→DOING→DONE), ale grid je řádkový. Bez reading-flow Tab skáče chaoticky po řádcích:
Tab pořadí = DOM pořadí → skáče přes sloupce (T1→D1→X1→T2→D2→X2→T3→D3→X3):
TODO
DOING
DONE
Klikni na kartu nebo zmáčkni Tab…
/* reading-flow — Tab pořadí sleduje vizuální rozvržení */

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  reading-flow: grid-order;  /* Chrome 137+ */
}

/* Bez reading-flow: Tab = DOM pořadí, může skákat po obrazovce */
/* S reading-flow: Tab = vizuální pořadí v gridu               */

/* Totéž pro flex (flex-visual sleduje vizuální pořadí) */
.flex-reversed {
  display: flex;
  flex-direction: row-reverse;  /* vizuálně: C B A */
  reading-flow: flex-visual;    /* Tab: C→B→A (ne A→B→C) */
}

/* Proč záleží na pořadí?
   – Uživatelé klávesnice procházejí UI zleva doprava, shora dolů
   – Screen readery čtou ve stejném pořadí
   – Nesprávné pořadí = zmatené UX, WCAG 2.4.3 Focus Order */

/* ─── Kanban board — DOM po sloupcích, grid po řádcích ─── */

/* DOM pořadí: T1, D1, X1, T2, D2, X2 … (seřazeno po sloupcích)
   Grid ale umísťuje karty explicitně do buněk (grid-area)
   → vizuální pořadí jde po řádcích: T1→D1→X1→T2→D2→X2
   → Tab bez reading-flow jde po DOM: T1→D1→X1→T2→D2→X2 ✓
     ale pokud chceš sloupce: T1→T2→T3→D1→D2→D3, přidej reading-flow */

.kanban-board {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto repeat(3, auto);
}

/* Karty umístěny explicitně: grid-area: řádek/sloupec */
/* T1 */ .kanban-card:nth-child(1) { grid-area: 2/1; }
/* D1 */ .kanban-card:nth-child(2) { grid-area: 2/2; }
/* X1 */ .kanban-card:nth-child(3) { grid-area: 2/3; }

/* S reading-flow: Tab sleduje vizuální grid pořadí */
.kanban-board.use-reading-flow {
  reading-flow: grid-order;
  /* Tab: T1→D1→X1→T2→D2→X2→T3→D3→X3 (po řádcích, zleva doprava)
     nebo po sloupcích pokud DOM odpovídá grid-order */
}
